# C-ADTs
A collection framework in C for practicing abstract data types and usable for algorithm development.

See type.h for the arbitrary type definition (union of primitive types)

Lists:
  - Linked List
  - Array List

Sets:
  - Tree Set
  - Hash Set

Heaps:
  - Min/Max Heap
  - Fibbonacci Heap

Trees:
  + Binary Search Tree
  + Red Black Tree

Maps:
  - Hash Map
